# 專案介紹
稅制在當今國家財政與個人財務規劃中扮演了關鍵角色。本專案先介紹台灣、中國、澳洲與日本的稅制規則，再透過Python撰寫累進稅率模型，並以圖表等方式展示稅賦計算情形。

### 專案目標：
- 分析不同收入階層在各種稅制下的稅賦負擔。
- 幫助用戶理解累進稅率的運作方式與扣除項目影響。
- 能透過程式計算預期稅額。
- 轉換成台幣，比較各國稅額。

# 功能介紹
- 支援多國稅制分析和計算（台灣、中國、澳洲、日本）。
- 採用累進稅率進行稅額計算，包含多種扣除項目與免稅額。
- 提供淨所得視覺化圖表，便於比較不同收入階層的稅賦分配。
- 以 Python 撰寫，可擴展為其他國家的稅制分析。
